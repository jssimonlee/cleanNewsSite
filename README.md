# cleanNewsSite

뉴스 기사 페이지를 읽기 좋게 정리해서 새 탭으로 열어주는 북마크릿 프로젝트입니다.

## 구성

- `reader-mode-bookmarklet-source.js`
  - 수정용 원본 소스입니다.
- `reader-mode-bookmarklet.js`
  - 브라우저 북마크 URL에 바로 넣는 북마크릿 파일입니다.

## 사용 방법

1. `reader-mode-bookmarklet.js` 파일 내용을 엽니다.
2. 전체 내용을 복사합니다.
3. 브라우저에서 새 북마크를 만듭니다.
4. 북마크 URL 칸에 붙여넣습니다.
5. 뉴스 기사 페이지에서 해당 북마크를 클릭합니다.

## 기능

- 기사 제목과 본문 위주로 추출
- 불필요한 링크, 댓글, 관련 기사 영역 최소화
- 글자 크기 조절
- 기사 읽어주기(TTS)
- 링크 보기 토글

## 참고

- 사이트 구조에 따라 일부 기사에서는 본문 추출 결과가 다를 수 있습니다.
- 수정은 `reader-mode-bookmarklet-source.js`에서 하고, 필요하면 다시 북마크릿 파일을 생성해서 사용합니다.
